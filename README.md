# 🤖 SigmaGPT - AI Chat Application

![SigmaGPT Banner](https://img.shields.io/badge/Project-AI%20Chatbot-blue)

## 📌 Project Overview

SigmaGPT is an AI-powered chatbot application that provides intelligent conversations using Large Language Models (LLMs). It allows users to interact with an AI assistant through a modern web interface.

The project is developed using the MERN stack and integrates AI APIs to generate human-like responses.

---

## 🚀 Features

* 💬 Real-time AI conversations
* 🤖 Intelligent chatbot responses
* 📝 Chat history management
* 🔐 Secure backend API
* 🌐 Modern responsive user interface
* ⚡ Fast and interactive experience

## 🎯 Application Workflow

1. User enters a message.
2. Frontend sends request to backend.
3. Backend communicates with AI API.
4. AI generates response.
5. Response is displayed to the user.
